# tax-engine

## What this is
Pure Java VAT rule engine. Implements deterministic tax calculations and classification logic using types from `core-domain`.

## Prerequisites
- JDK 21
- Gradle wrapper

## How to build
```bash
./gradlew :tax-engine:build
```

## How to run locally
This module is a library and has no standalone runtime.

## How to run tests
```bash
./gradlew :tax-engine:test
```

## Environment variables
| Variable | Required | Default | Description |
